Execute

Execute plugin of APData from the following list. Plugin in list should be selected! This operation should open configuration frame of data plugin or execute plugin process.

Display

Display selected data from the following list.

as group

Two or more below selected data is displayed as group.

Change

Change plotter of below selected data on plotter selected in next combo box.

Configure

Open configuration frame of plotter which is attached to below selected position.

Hide

Remove below selected data from the following list.
