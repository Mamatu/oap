2. Chart frame.

Save as - save chart screen to image

Menu:

  • Auto refreshing - automatically refresh chart after some change in dynamic data
  • Auto adjusting - automatically adjust this chart to display whole dynamic data.
  • Auto shifting - automatically shifting chart to track dynamic data
  • Use dynamic step - use this step to create ticks instead step set in Properties Frame -> x/y ticks

Rec - record change on chart

Stop - stop record change on chart

Continue - continue record change on chart

Save - save recorded charts to images

Setup - set up options

3. Properties frame.

x/y range - scope of x/y axis [Min:Max]

x/y ticks - set up which ticks should be visible on chart. [Min:Max:Step;Value1;Value2] - you can create ticks from Min to Max and use Step, you can set individual tick if put value in semicolons.

x/y length of label - length of string which will be visible near x/y axis on chart.

x/y dynamic step - step which will be used if you select option Chart Frame -> Menu -> Use dynamic step

4. Data frame.

Execute - execute plugin which is selected bottom

Display - display data which is selected bottom

Change - change shape which is used to plot data (and which is selected in compbo box)

Hide - stop displaying data which is selected bottom

Configure - show configuration frame of displayed shape

< - previous elemnt of data history

> - next element of data history

Return - return to current state of data

Use for all repositories - use action of three buttons near this check box to apply change to all data which is in 'Displayed data'
