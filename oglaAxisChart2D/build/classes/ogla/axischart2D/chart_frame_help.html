Chart frame.

Save as - save chart screen to image

Menu:

  • Auto refreshing - automatically refresh chart after some change in dynamic data
  • Auto adjusting - automatically adjust this chart to display whole dynamic data.
  • Auto shifting - automatically shifting chart to track dynamic data
  • Use dynamic step - use this step to create ticks instead step set in Properties Frame -> x/y ticks

Recording panel.

It allow you to record changes on chart. If chart is refreshed/updated, charts are save to container. If you want to you can save recorded images of chart to some directory as set of pictures or to images container (this options are not working now). Befor saving needed is set up options (Setup button).

Record - record changes on chart

Stop - stop record change on chart

Continue - continue record change on chart

Save - save recorded charts to images

Setup - set up options
