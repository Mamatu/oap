3. Properties frame.

x/y range - scope of x/y axis [Min:Max]

x/y ticks - set up which ticks should be visible on chart. [Min:Max:Step;Value1;Value2] - you can create ticks from Min to Max and use Step, you can set individual tick if put value in semicolons.

x/y length of label - length of string which will be visible near x/y axis on chart.

x/y dynamic step - step which will be used if you select option Chart Frame -> Menu -> Use dynamic step
